Introduction

This document provides instructions for deploying the React app on a web server. These instructions assume that you have already installed Node.js and have access to a web server.

Step 1: Build the App

Open a command prompt and navigate to the root directory of the React app. Run the following command to build the app:

npm run build

This will create a production build of the app in the build directory.

Step 2: Copy Files to Web Server

To deploy the React application, you need to copy the build files to a web server that will serve them to the users. Follow these steps to copy the files to the server:

  1. Open your preferred FTP client.
  2. Connect to your web server using the provided login credentials.
  3. Navigate to the root directory of your web server.
  4. Create a new folder named my-react-app in the root directory of your web server.
  5. Navigate to the build folder in the React project directory.
  6. Select all files and folders in the build folder and drag them to the my-react-app folder in your FTP client.
  7. Wait for the files to finish copying to the server.

Step 3: Test the Deployment

Now that you have deployed the React application to your web server, you can test the deployment by accessing the application in a web browser. Follow these steps to test the deployment:

  1. Open a web browser.
  2. Enter the URL for your deployed React application in the address bar. For example: http://example.com/my-react-app.
  3. Press Enter.
  4. The React application should load in the web browser. Verify that the application is functioning correctly and that all features are working as expected.

Step 4: Install Dependencies

Install the dependencies needed for the app by running the following command:

npm install

Step 5: Run the App

To run the app, execute the following command:

npm start

Step 6: View the App

Open a web browser and go to http://localhost:3000 to view the app.

Results

The app should now be running and can be viewed at http://localhost:3000.
